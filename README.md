# EEG Privacy Analysis

This repository contains code, notebooks, results, and reports for EEG privacy and utility experiments.

## Goals

- Evaluate EEG model utility.
- Measure subject-identification privacy leakage.
- Compare different train/test splitting strategies.
- Study the privacy-utility trade-off.

## Structure

- `notebooks/`: Jupyter/Colab notebooks
- `src/`: reusable Python code
- `results/`: experiment result CSVs
- `figures/`: plots and visualizations
- `reports/`: summaries and PDFs
- `data/`: dataset notes only; raw data is not uploaded

## Data

Raw EEG datasets are not included because of size, privacy, and licensing concerns.
